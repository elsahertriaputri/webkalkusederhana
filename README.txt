PETUNJUK PENGGUAAN KALKULATOR SEDERHANA

1. Masukan bilangan pertama dan bilangan kedua pada
   kolom yang tersedia, dapat berupa bilangan bulat
   atau pecahan. Namun, kotak yang tersedia hanya dapat 
   menerima input berupa karakter numerik.
2. Gunakan titik (.) sebagai tanda desimal. 
3. Tekan salah satu operasi matematika yang hendak
   digunakan untuk penghitungan.
4. Hasil perhitungan akan muncul seketika saat tombol
   operasi ditekan. Pembagian dengan bilangan nol (0)
   akan menghasilkan error.
5. Hapus dan ganti bilangan pada kotak input untuk
   memulai penghitungan baru.
